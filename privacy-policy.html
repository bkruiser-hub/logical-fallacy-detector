Privacy Policy

Logical Fallacy Detector - Chrome Extension
Last updated: March 28, 2026

Summary: We do not collect, store, or transmit any of your personal data. Your API key stays on your device. Page content is sent directly from your browser to Google's API.

1. Overview

Logical Fallacy Detector ("the Extension") is a Chrome browser extension that analyzes webpage content for logical fallacies using Google's Gemini AI. This privacy policy explains how the Extension handles your data.

2. Data We Collect

We collect no data whatsoever. The Extension:

  • Does NOT collect personal information
  • Does NOT use analytics or tracking
  • Does NOT use cookies
  • Does NOT have user accounts
  • Does NOT have a server or backend

3. Data Stored Locally

The Extension stores the following locally on your device using Chrome's storage API:

  • Your Google Gemini API key - stored securely so you don't have to re-enter it. This key never leaves your browser except when making API calls directly to Google.
  • Your image analysis preference - whether the "Include images" toggle is on or off.

4. Data Transmitted to Third Parties

When you click "Analyze This Page," the Extension sends the following directly from your browser to Google's Generative Language API:

  • Text content of the webpage (up to ~10,000 characters)
  • Up to 5 images from the page (if image analysis is enabled), resized to max 512px
  • Your Gemini API key (for authentication)

This data goes directly from your browser to Google. It does not pass through any server we operate. Google's use of this data is governed by Google's Privacy Policy.

5. Permissions Explained

  • activeTab - Read content of the tab you're viewing when you click the extension icon
  • storage - Save your API key and preferences locally
  • scripting - Extract text and images from the current page
  • host_permissions - Run content extraction on any website you choose to analyze

6. Data Retention

The Extension does not retain any webpage content after analysis. Once results are displayed, extracted text and images are discarded from memory.

7. Children's Privacy

The Extension is not directed at children under 13.

8. Changes to This Policy

If we update this policy, we will update the "Last updated" date above.

9. Contact

Questions? Contact us through the Chrome Web Store support tab for this extension.
